# usda-food-nutrient-analysis
Excel-based nutrient density analysis using USDA FoodData Central

Nutrient Density Analysis of Branded Foods
Using USDA FoodData Central
Project Overview
This project analyzes branded food products from the USDA FoodData Central database to evaluate nutrient density across food categories. The goal is to move beyond serving-based nutrition labels by standardizing nutrients per gram and identifying calorie-dense, protein-dense, and sodium-heavy food categories.
